List of Files in the Directory:

1. anothergame.py  
	       - Python file containing the simulation code. The file contains the detailed explanation for the functions and the interface description. 

2. Probabilities Worksheet.xlsx and Probabilities Worksheet.xls (copy - for compatibility) 
	       - Excel files containing Transition Probability Matrix calculations and Sample manual simulations.

3. Readme (this file) 
               - A desription of files in the .zip folder.
************************************************************

How to Run the code:

1. Open a Python console (Python version 3.7 or later) and navigate to the directory containing the file "anothergame.py".
2. Run the command >python anothergame.py
3. Results will be displayed and the histogram will be downloaded as image in the same directory.
